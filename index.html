Airport Solver Flights:

Languages: VB.NET

This program calculates every possible shortest path between all airports designated in the program files. All the user must do is select a maximum travel radius, which would limit how far each flight can be. The shorter the radius, the more arcs and nodes will be generated. Once the program is run, the GUI displays all airports, as well as a dropdown of all other airports in the system. When the destination airport is clicked, the shortest path (layovers if necessary), capacity, and cost are revealed in a textbox below.

Course: BIT 3444 - Advanced Business Computing and Applications
